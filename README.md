# GenAI-Phys-Collab

Scripts for model preparation and examples with COMSOL scattering 2d-axisymmetric simulations for optimization task.

### Setup
In this task, incoming light is TE-polarized plane electromagnetic wave with wavelength 1550 nm propagating in the positive z-direction at angle of incidence 30 degrees. 
The structure is composed of 10 cylindrical layers with different radii around 300 nm and total height 800 nm. 
The target scattered coefficients are generated by solving COMSOL axisymmetric scattering problem with some random geometry of the structure.
To obtain the target multipole coefficients, the scattred field is decomposed into outgoing Spherical Vector Wave Functions (SVWFs) using the method described in [*]:
$$
\mathbf{E}_{\text{scat}}(\mathbf{r}) = \sum_{n=1}^{N} \sum_{m=-n}^{n} a_{nm} \mathbf{M}_{nm}(k\mathbf{r}) + b_{nm} \mathbf{N}_{nm}(k\mathbf{r})
$$
In this task $N=4$, so there are 24 + 24 = 48 complex multipole coefficients in total in `target.pkl`, where the structure of the coefficients is as follows:
```python
{
    "a": {
        1: {
            -1: a_{1,-1},
            0: a_{1,0},
            1: a_{1,1},
        }
        2: {
            -2: a_{2,-2},
            -1: a_{2,-1},
            0: a_{2,0},
            1: a_{2,1},
            2: a_{2,2},
        },
        ...
        N: {
            -N: a_{N,-N},
            ...,
            N: a_{N,N},
        }
    },
    "b": {
        1: {
            -1: b_{1,-1},
            0: b_{1,0},
            1: b_{1,1},
        }
        ...
        N: {
            -N: b_{N,-N},
            ...,
            N: b_{N,N},
        }
    }
}
```
Prepared COMSOL model `models/model.mph` should be used as a base model and solver for optimization. 

### Navigation
- `workflow.ipynb`: workflow example for solving the model and exporting multipole coefficients.
- `target.pkl`: generated target multipole coefficients.
- `2d_mult_dec.mph`: base COMSOL model* from https://github.com/Sag050196/multipole-decomposition-and-second-harmonic-generation-using-2D
- `src/prepare_model.py`: loads `2d_mult_dec.mph`, clears results, rebuilds polygon geometry, and saves `model.mph`. Use `uv run python prepare_model.py` to prepare the model.
- `models/model.mph`: prepared for optimization COMSOL model.
- `models/target.mph`: COMSOL model with geometry corresponding to the target multipole coefficients.
- `src/target_model.py`: script for preparing `target.mph` from `models/model.mph`.
- `src/cylinder.py`: creation of polygon geometry.
- `src/ab_multipoles.py`: evaluating multipole coefficients from COMSOL results.
